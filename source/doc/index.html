What is Mootor?

Mootor is an open source framework for building mobile applications based on HTML5.

HTML5 because it’s a fast growing technology and it’s multi-platform (Android, iOS, Firefox OS, etc). Open source because a front-end tool in HTML5 can not avoid it.

Keyboard shortcuts

Browse to a module or class using the sidebar to view its API documentation.

  • Press s to focus the API search box.

  • Use Up and Down to select classes, modules, and search results.

  • With the API search box or sidebar focused, use ⌘-Left or ⌘-Right to switch sidebar tabs.

  • With the API search box or sidebar focused, use Ctrl+Left and Ctrl+Right to switch sidebar tabs.
